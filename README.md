# 🔐 Secure File Encryptor (Java)

## 📌 Overview
This project implements AES-based file encryption and decryption using Java.
It allows users to securely encrypt and decrypt files using symmetric encryption.

---

## 🚀 Features
- AES key generation
- File encryption
- File decryption
- Clean modular structure

---

## 🛠️ Tech Stack
- Java
- javax.crypto library
- Git & GitHub

## 🔑 What is AES?
AES (Advanced Encryption Standard) is a symmetric encryption algorithm widely used for secure data storage and transmission.
